《非暴力沟通 5》读书笔记

马歇尔·卢森堡 沟通与关系终极版 2026-03-10

● 核心定位

用爱与理解重建沟通的桥梁

《非暴力沟通 5》是关于沟通与关系的经典之作。本书的核心洞见：大部分冲突源于表达方式，不是事实本身。非暴力沟通通过观察、感受、需要、请求四个要素，建立连接而非对抗。学会非暴力沟通，就能化解冲突，建立深层关系。

● 非暴力沟通四要素

四要素详解

要素核心表达方式
1. 观察事实而非评判"我看到/听到..."
2. 感受表达情绪"我感到..."
3. 需要深层需求"因为我需要..."
4. 请求具体行动"你愿意...吗？"

暴力沟通 vs 非暴力沟通

维度暴力沟通非暴力沟通
关注点对错、评判感受、需要
表达指责、命令请求、连接
结果对抗、防御理解、合作
关系疏远、伤害连接、信任

● 核心方法

表达愤怒的四步骤

  1. 停下来呼吸：暂停，不立即回应
  2. 识别评判：识别自己的评判想法
  3. 连接需要：识别愤怒背后的需要
  4. 表达感受和需要：用非暴力方式表达

倾听他人的四步骤

  1. 放下已有想法：清空自己的判断
  2. 体会对方感受：同理心感受
  3. 理解对方需要：看到深层需要
  4. 反馈确认：确认理解正确

请求的艺术

  • 具体：请求具体行动，不是抽象
  • 正向：请求做什么，不是不做什么
  • 可拒绝：允许对方拒绝
  • 不命令：是请求，不是命令

● 核心观点

七大核心洞察

1. 评判是暴力之源

评判别人，引发防御和对抗。放下评判，才能连接。

2. 感受揭示需要

感受是指南针，指向未满足的需要。识别感受，找到需要。

3. 所有行为都为了满足需要

即使是破坏性行为，也是为了满足需要。找到需要，才能理解行为。

4. 愤怒是信号

愤怒不是别人造成的，是自己的需要未满足。愤怒是指向需要的信号。

5. 同理心是核心

同理心是理解他人感受和需求。先同理，才能被理解。

6. 请求不是命令

命令引发反抗，请求创造合作。允许拒绝，才能建立信任。

7. 爱自己是非暴力基础

对自己非暴力，才能对他人非暴力。原谅自己，接纳自己。

解析：先同理他人，才能被理解。同理心建立连接。

● 实践应用

场景 1：夫妻冲突

  • 观察：描述事实，不是评判
  • 感受：表达真实感受
  • 需要：揭示深层需要
  • 请求：提出具体请求

场景 2：亲子沟通

  • 倾听：同理心倾听孩子
  • 理解：理解孩子的感受
  • 引导：帮助孩子识别需要

场景 3：职场沟通

  • 反馈：用观察和感受，不是评判
  • 协作：用请求，不是命令
  • 理解：理解同事的需要

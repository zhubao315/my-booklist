《巴菲特的投资原则》读书笔记

沃伦·巴菲特#价值投资#长期主义

● 作者介绍

沃伦·巴菲特，伯克希尔·哈撒韦公司CEO，被誉为"奥马哈先知"。通过价值投资从100美元起步，成为世界顶级富豪。他的投资哲学影响了全球数代投资者。

● 核心框架

🎯 巴菲特投资四原则

  1. 买股票就是买公司：关注企业本身，而非股价波动
  2. 市场先生是仆人：利用市场情绪，而非被其左右
  3. 安全边际：以低于内在价值的价格买入
  4. 能力圈：只投资自己理解的企业

● 核心观点

  1. 长期持有：最喜欢的持有期限是永远
  2. 护城河：寻找具有持久竞争优势的企业
  3. 管理层：优秀的管理层是企业的灵魂
  4. 简单易懂：投资自己能理解的业务
  5. 集中投资：把鸡蛋放在少数几个篮子里，然后看好它们

● 金句摘录

"价格是你付出的，价值是你得到的。"

解析：区分价格和价值是投资的核心。好公司也可能成为坏投资，如果价格太高。

"在别人贪婪时恐惧，在别人恐惧时贪婪。"

解析：逆向思维的经典表达。市场情绪是投资者最好的朋友或最大的敌人。

"投资的第一条规则是不要亏损，第二条规则是永远不要忘记第一条。"

解析：风险控制永远是第一位的。避免永久性损失比追求高收益更重要。

"如果你不愿意持有一只股票10年，那就不要持有它10分钟。"

解析：长期主义是价值投资的核心。短期波动是噪音，长期价值是信号。

● 应用指导

场景 1：选股清单

📊 巴菲特式选股五问

  1. 我能理解这个业务吗？ → 能力圈
  2. 它有持久的竞争优势吗？ → 护城河
  3. 管理层是否诚实能干？ → 治理
  4. 价格是否合理？ → 安全边际
  5. 我会愿意持有10年吗？ → 长期主义

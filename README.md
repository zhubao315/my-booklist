## 🚀 阅读策略

### 🎯 按需选择指南

- **需要解决问题** → 实用技能类：精益创业、定位、增长黑客
- **追求自我成长** → 认知成长类：思考，快与慢、原子习惯、自控力
- **理解社会运行** → 历史社科类：人类简史、国富论、乌合之众
- **把握科技趋势** → 科技科普类：必然、失控、浪潮之巅
- **建立财富观念** → 投资理财类：聪明的投资者、穷查理宝典、原则

### 📅 场景化阅读建议

- **通勤碎片时间**：认知成长、科技科普类（15-20分钟）
- **周末深度阅读**：历史社科、哲学宗教类（2-3小时）
- **工作学习间隙**：实用技能、技术类（配合实践项目）
- **投资理财规划**：投资理财、金融类（系统性学习）

### ⭐ 精读方法论

1. **每类前5本为核心必读** - 建立该领域的基础框架
2. **从最重要的开始** - 优先阅读影响力最大、实用性最强的作品
3. **深度内化而非读完** - 真正的价值在于吸收和实践
4. **建立知识连接** - 将不同领域的知识进行有机整合
